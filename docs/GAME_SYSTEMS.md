# Nexus Chronicles: Game Systems Overview

This document provides a brief overview of some core game mechanics.

## Character Creation: Point-Buy System

Character attributes (Strength, Dexterity, etc.) are determined using a **27-point buy system**.

-   Each stat starts at a base of **8**.
-   Players have **27 points** to spend to increase these stats.
-   The cost to increase a stat is not linear. Higher scores cost more points.

### Point Costs

| Stat Score | Point Cost |
| :--------: | :--------: |
|     8      |     0      |
|     9      |     1      |
|     10     |     2      |
|     11     |     3      |
|     12     |     4      |
|     13     |     5      |
|     14     |     7      |
|     15     |     9      |

The maximum score a player can buy is **15**. Racial bonuses are applied *after* points are spent.

## Karma System

Karma is a hidden stat that tracks the player's moral alignment based on their actions and choices throughout the game.

-   **Gaining Karma:** Positive karma is gained through selfless, honorable, or kind actions, such as donating to the temple.
-   **Losing Karma:** Negative karma is acquired through criminal, cruel, or selfish actions, such as accepting a job from the Shadow Syndicate.

### Effects of Karma

A player's karma score can unlock special dialogue options or change how certain NPCs react to them.

-   **Example:** If the player has negative karma, Brom the barkeep may become warier and less forthcoming with information.
